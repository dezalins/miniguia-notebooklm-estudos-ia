# 📌 About the Project

This project explores how **Artificial Intelligence can be used as a structured learning tool** to synthesize knowledge from multiple sources.

Using **NotebookLM**, a thematic notebook was created to study the science behind:

- strength training
- running performance
- fat loss
- muscle hypertrophy
- hybrid training

The project demonstrates how **AI-assisted knowledge extraction** can transform multiple sources into a **structured study system**.

Notebook used in this project:

https://notebooklm.google.com/notebook/06ab90b3-e4d0-4942-8d61-ec131c12beed

---

# 🎯 Research Question

How can strength training and endurance training be combined efficiently to promote **fat loss while gaining muscle**?
