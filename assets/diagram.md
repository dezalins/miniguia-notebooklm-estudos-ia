# AI Learning Workflow Diagram

This diagram represents the workflow used in this project to transform raw learning material into structured knowledge using AI.

Sources → NotebookLM → Prompt Engineering → Insights → Study Guide

## Workflow Explanation

1. **Curated Sources**  
   Learning materials related to strength training, running, and exercise physiology were selected.

2. **NotebookLM Knowledge Base**  
   The sources were uploaded to NotebookLM to create a contextual AI knowledge environment.

3. **Prompt Engineering**  
   Structured prompts were used to guide AI analysis and extract deeper insights.

4. **AI Knowledge Extraction**  
   NotebookLM synthesized information across multiple sources.

5. **Iterative Questioning**  
   Follow-up prompts were used to refine and deepen the analysis.

6. **Insights Generation**  
   The final output included structured knowledge, training insights, and a practical training plan.
